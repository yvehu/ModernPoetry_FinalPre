# 本地演示指南

## 快速启动

### 1. 安装依赖（如果还没有）

```bash
cd /Users/yve/Desktop/ModernPoetry_FinalPre
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器会自动打开浏览器，通常地址是：`http://localhost:3000`

### 3. 允许摄像头权限

- 浏览器会弹出摄像头权限请求
- 点击"允许"或"Allow"
- 如果错过了，点击地址栏的锁图标 → 摄像头 → 允许

### 4. 使用手势控制

- **张开/合拢手掌**：控制粒子缩放
- **切换场景**：点击左下角的场景按钮

## 演示前检查清单

- [ ] 依赖已安装：`npm install`
- [ ] 开发服务器运行：`npm run dev`
- [ ] 浏览器已打开：`http://localhost:3000`
- [ ] 摄像头权限已允许
- [ ] 摄像头没有被其他应用占用
- [ ] 光线充足（手势识别需要）

## 演示技巧

### 最佳环境
- ✅ 光线充足
- ✅ 背景简洁（避免复杂背景）
- ✅ 摄像头位置稳定
- ✅ 手部清晰可见

### 手势操作
- **放大**：张开手掌，手指分开
- **缩小**：合拢手掌，手指并拢
- **切换场景**：点击左下角按钮

### 如果手势不灵敏
- 确保手部完全在摄像头视野内
- 尝试调整手与摄像头的距离
- 确保光线充足
- 避免背景过于复杂

## 停止服务器

按 `Ctrl + C` 停止开发服务器

## 故障排除

### 端口被占用？

如果 3000 端口被占用，Vite 会自动使用其他端口（如 3001），查看终端输出确认实际端口。

### 摄像头无法访问？

1. 检查系统摄像头权限
2. 确保摄像头没有被其他应用占用
3. 尝试重启浏览器
4. 检查浏览器控制台是否有错误

### 页面空白？

1. 检查终端是否有错误
2. 确认所有依赖都已安装
3. 尝试清除浏览器缓存

## 演示流程建议

1. **开场**：介绍项目背景和四个季节场景
2. **Spring（春天）**：展示粉色樱花，演示手势缩放
3. **Summer（夏天）**：切换到夏天场景，展示粉蓝色晚霞和橙色倒影
4. **Autumn（秋天）**：展示棕色叶子，演示缩放效果
5. **Winter（冬天）**：展示复杂雪花，演示最终效果
6. **总结**：强调交互性和技术实现

## 备用方案

如果摄像头在演示时出现问题：
- 可以提前录制演示视频
- 或者使用屏幕录制软件实时演示
- 准备截图作为备用展示
