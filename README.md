# Four Ways of Decoding New York - 3D 粒子系统

一个基于 React + Three.js + MediaPipe 的实时交互式 3D 粒子系统，对应诗歌的四个季节场景。

## 🚀 快速开始（本地演示）

### 方法一：使用启动脚本（推荐）

```bash
./start-demo.sh
```

### 方法二：手动启动

```bash
# 1. 安装依赖（首次运行）
npm install

# 2. 启动开发服务器
npm run dev
```

服务器会自动打开浏览器，地址通常是：**http://localhost:3000**

## 🎮 使用说明

### 手势控制
- **张开手掌**：放大粒子模型
- **合拢手掌**：缩小粒子模型
- **切换场景**：点击左下角的场景按钮

### 四个场景
1. **Spring（春天）** 🌸 - 粉色樱花
2. **Summer（夏天）** 🌅 - 粉蓝色晚霞和橙色倒影
3. **Autumn（秋天）** 🍂 - 棕色叶子
4. **Winter（冬天）** ❄️ - 白色雪花

- **React** - UI 框架

## 🐛 故障排除

### 摄像头无法访问？
- 检查浏览器权限设置
- 确保摄像头没有被其他应用占用
- 查看浏览器控制台错误信息

### 手势识别不灵敏？
- 确保光线充足
- 手部完全在摄像头视野内
- 调整手与摄像头的距离

### 端口被占用？
- Vite 会自动使用其他端口
- 查看终端输出确认实际端口号
